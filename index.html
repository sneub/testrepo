Getting Started

Hey! Let me help you.

If you've never done this before, it may seem a little daunting. Don't worry! I'm here for you. The guide below will step you through the whole project set up process and have you querying the AdWords API in no time.

Google Developer Console

The first thing we need to do is open the Developer Console for your project

Open Script editor from the Tools menu

Open Developers Console Project from the Resources menu

Click View Developers Console to open the console

Navigate to the Credentials section

Click on the Apps Script link to see credentials related to Apps Script

Find your Client ID and Client Secret here

  • Take note of the Client ID and Client Secret here.
  • Insert the Authorized Redirect URL in this page too.
  • Don't forget to save your changes

Now you're ready to authorize and start querying the AdWords API!
